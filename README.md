# Boston Weather Dashboard (Matplotlib Project)

This project is a real-time data visualization exercise using **Python**, **NumPy**, **Pandas**, and **Matplotlib**.

It visualizes **14 days of actual Boston-style weather data**, including:
- Daily temperature (high/low)
- Daily humidity
- Weather description (sunny, cloudy, snow, etc.)

The purpose of this project is to practice:
- Loading data from a CSV
- Creating visualizations using Matplotlib
- Using subplots, annotations, titles, labels, and styling
- Computing and plotting averages
- Highlighting min/max temperature points
- Saving figures for reports or LinkedIn posts
